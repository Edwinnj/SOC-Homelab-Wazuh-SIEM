# SOC Homelab - Wazuh SIEM

This project demonstrates a cybersecurity homelab used to simulate attacks and detect them using a SIEM.

## Architecture

The lab contains two machines:

- Kali Linux (Attacker)
- Ubuntu Server running Wazuh SIEM

## Attack simulation

The attacker performs reconnaissance and brute force attacks.

### Tools used

- Nmap
- Hydra
- Wazuh SIEM

## Example attack

SSH brute force attack using Hydra.

## Detection

Wazuh detects multiple failed login attempts.

## MITRE ATT&CK

Technique: T1110 - Brute Force
